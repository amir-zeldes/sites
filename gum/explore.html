What can you do with GUM annotations?

The annotations in GUM's data model allow access to deep semantic and pragmatic information about what each document is talking about, why things are said, and which parts are more prominent than others. Documents analyzed with these types of annotations allow us to do much more than passive reading in a normal browser document:

  • Semantic find - ever wish you could search for all mentions of some person or place in a document, regardless of how they are referred to?
  • Main points - using discourse annotation graphs we can:
    • Jump to the main points of the entire text
    • Highlight the core or 'nucleus' part of each paragraph, possibly hiding the rest of the text for speed reading
    • Restrict semantic searches just to main points (e.g. entities occurring in a paragraph nucleus)
  • Summary of events - get an overview of everything someone or something says or does in a text
  • Search by function - find all parts of a text supplying evidence, giving attribution for sources of statements, or conceding something which contradicts what the text says
  • Entity linking - know whether there is a Wikipedia article for each entity in the text, and if so, get a link to read more about it
